Barre de champs (FieldBar)

Les barres de champs permette de créer facilement un visuel à partir d'une ligne de texte.

Usage

Le visuel CSS/HTML est totalement généré depuis un champ texte. Le caractère de séparation ';' est utilisé pour diviser la chaîne en champs. Chaque champ représente un élément HTML.

On peut facilement imaginer les applications utiles d'une Barre de champs:

  • Gérer une liste de contact dans un e-mail
  • Créer une barre de navigation dynamique
  • Lister des options dans un formulaire
