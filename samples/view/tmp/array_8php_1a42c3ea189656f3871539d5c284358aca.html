array_to_tree

Convertie un tableau simple en tableau d'arborescence.

Syntaxe

array_to_tree(
[in] $simple_array,
);

Paramètres

$simple_array
Tableau à convertir

Retourne

Tableau en arborescence
[array] : Tableau en arborescence

Remarques

Chaque clé du tableau correspond à un chemin de noms séparé par des points (.)

Exemple

Utilisation de la fonction array_to_tree() $ar = array( "ceci.est.un_chemin" => "ceci est la valeur 1" , "ceci.est.un_autre_chemin" => "ceci est la valeur 2" ); print_r( array_to_tree($ar) ); Sortie: Array ( [ceci] => Array ( [est] => Array ( [un_chemin] => ceci est la valeur 1 [un_autre_chemin] => ceci est la valeur 2 ) ) )

array_cmp

Compare deux tableaux.

Syntaxe

array_cmp(
[in] $ar1,
[in] $ar2,
);

Paramètres

$ar1
Tableau à comparer
$ar2
Tableau à examiner

Retourne

Différence entre les deux tableaux
[false] : une ou plusieur clés de $ar1 est introuvable dans $ar2
[true] : toutes les clés de $ar1 sont existantes dans $ar2
